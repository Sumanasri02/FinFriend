# 💸 FinBuddy

A full-stack personal finance tracker app.

## 🛠 Tech Stack

- **Frontend**: Angular 19, Chart.js, ng2-charts  
- **Backend**: Node.js, Express.js, MongoDB, Mongoose  
- **Database**: MongoDB (Local)

## 📂 Folder Structure


## 🚀 Running the Project

### Backend
cd finbuddy-backend
npm install
npm start

### Frontend
cd finbuddy-frontend
npm install
ng serve
